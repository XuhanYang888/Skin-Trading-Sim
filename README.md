### Project Description

An app built to track a specific game (undecided, possibly CS2 or TF2) on the Steam Market. The engine scrapes market data over a long time to build a database of item histories. By training a multi-factor forecasting models, it can project price trajectories and flag market anomalies. There will be an interactive dashboard that lets you review market insights and manage a simulated inventory to experience being a skin trader in a risk-free simulated environment.
